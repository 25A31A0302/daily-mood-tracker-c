# Daily Mood Tracker in C

A simple command-line application built in C to record and analyze daily moods.

## Features

- Add moods
- View mood history
- View mood statistics
- Simple menu-driven interface

## Concepts Used

- Arrays and Strings
- Functions
- Loops
- Conditional Statements
- User Input Handling

## How to Run

Compile:

```bash
gcc mood_tracker.c -o mood_tracker
```

Run:

```bash
./mood_tracker
```
